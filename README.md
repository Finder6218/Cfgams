# 燃气开户项目收费管理系统

《燃气开户项目收费管理系统》是提供燃气公司开户收费的一款软件,能提高收费效率、收费金额汇总。   
系统包含：  
1、基础设置：用气设置、复核设置、收款设置、项目设置  
 2、收费管理：收费登记  
  3、统计汇总：收费汇总
# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe"文件即可完整运行对应的信息管理系统。
